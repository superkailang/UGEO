# Installation

```
   CMake UGEO\src\CMakeLists.txt
```

# Usage
  
  Due to rendering the complex grid in 3D space, the program must run in the Windows system with at least one GPU.
  
  ```
  For Windows system with GPU
  
  Install Setup.exe
  
  Import Data from  DataSet/case
  ```
